# kanji-flash-card

A JLPT‑level based kanji flashcard web app with dark/light modes, featuring all 2,200 Jōyō kanji.

## Features
- Complete Jōyō kanji (2,200 characters) grouped by JLPT level (N5–N1)
- Flip cards with kanji (red), kun'yomi (blue), on'yomi (purple), meaning (magenta)
- Light mode (blue theme) and dark mode (purple theme) with persistent preference
- Shuffle, mark known/unknown, filter unknown, keyboard navigation
- Mobile‑responsive, no frameworks – just vanilla HTML/CSS/JS

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
   - No server required; all data is embedded.
3. Alternatively, serve the folder via a local web server if you prefer.
